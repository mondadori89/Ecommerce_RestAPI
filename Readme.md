# E-Commerce RestAPI 
A codecademy backend project.
<br />

## Install npm, express and postgres
Must have node installed

`npm i express pg ` <br />
`npm install express-session ` <br />
`npm install passport passport-local` -> not really need yet<br /> 
<br />

`npm start`  - to run it <br />
